# OneController (branch)

Single Controller Simple Delta Operation.

For use in demos where we only want to expo limited features with a single wireless controller.

## Controls

See DSC-Controlls.docx

## What is implemented?

* Teleop Drivebase
* Teleop Upper Mag
* Teleop Lower Mag
* Teleop Shooter
* Teleop Intake

## What is not implemented?

* Auto
* Automag
* Wheel of fortune
* Climber
* Fine Shooter
* Fine Hood
